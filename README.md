# Nancy Goel (Portfolio/Blog)

## Getting started

### Instalation and starting development server

```bash
# Install Gatsby CLI globally
yarn global add gatsby-cli

# Install dependencies
yarn

# Start development server
yarn develop
```

### Deploying to github-pages

```bash
yarn deploy:github
```
